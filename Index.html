What is VoidFarers

Voidfarers is the game where the player will be given a scenario where he is a Von Neumann Probe with the mission to create an empire in the stars for whatever country he chooses to represent. He is given a choice of background in various fields which give various buffs for resource gathering, fabrication, starting buffs, and etc. The game will end if the player starts on an ‘end narrative’ option which will be given out semi-early, and will be available to the player to activate whenever he wants to. One run of the game should be from 3 to 6 hours from start to finish.
